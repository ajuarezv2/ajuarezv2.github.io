Visualizations

How can data manipulations reveal a new insight into your dataset?

Data manipulations can help by creating a variable that is not already in your dataset. For ours, we added many, but to highlight one that we found the most useful was Netfilx Age, which told us how long a title has been on Netfilx’s catalogue. We used it to help us see how long a title is released before being added to Netflix and seeing if the trend has changed in recent years.

#quant vs quant
ggplot(netflix, aes(content_age, netflix_age, color = content_age)) +
  geom_point() +
  scale_x_log10() +
  scale_y_log10() +
  geom_smooth(method ="lm", color="red") + 
  labs( title= "Content Age vs Time on Netflix", x = "Content Age (Years)", y = "Years on Netflix")
## `geom_smooth()` using formula = 'y ~ x'
